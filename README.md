Informe Tecnico
El presente sistema de matrícula desarrollado en PHP permite registrar estudiantes en una base de datos MySQL mediante el uso de PDO y sentencias preparadas, garantizando seguridad ante inyecciones SQL. El script valida que la solicitud se realice por método POST, limpia los datos recibidos con `trim()`, verifica campos obligatorios (nombre, email y curso) y ejecuta una inserción segura en la tabla `estudiantes`. Además, implementa manejo de excepciones para controlar errores como registros duplicados y utiliza `htmlspecialchars()` para prevenir vulnerabilidades XSS al mostrar mensajes en pantalla. El sistema incluye retroalimentación visual diferenciando mensajes de éxito y error, asegurando una interacción clara con el usuario y cumpliendo con buenas prácticas básicas de desarrollo web seguro.
